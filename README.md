## 📋 四步检测法 | Four-Step Detection Method

### Step 1 — 查外观设计专利 | Design Patent Search

关键词检索 → 一键跳转全球专利平台：

| 平台 Platform | 覆盖范围 Coverage | 链接 |
|---|---|---|
| Google Patents | 全球 Global | [patents.google.com](https://patents.google.com/) |
| 国知局专利检索 CNIPA | 中国 China | [pss-system.cponline.cnipa.gov.cn](https://pss-system.cponline.cnipa.gov.cn/) |
| WIPO 外观设计数据库 | 国际 International | [wipo.int/designdb](https://www.wipo.int/designdb/) |
| EU DesignView | 欧盟 EU | [tmdn.org/tmdsview-web](https://www.tmdn.org/tmdsview-web/) |
| USPTO 外观专利 | 美国 US | [uspto.gov/patents/search](https://www.uspto.gov/patents/search) |

- 洛迦诺分类提示：包装设计主要涉及 **09类**（运输/包装容器）
- 第三方专业工具推荐：佰腾、Incopat、Patentics

### Step 2 — 查著作权 | Copyright Search

| 工具 Tool | 说明 Description |
|---|---|
| 中国版权保护中心 CCRC | 作品登记公告查询（需中国大陆网络） |
| 国家版权局 NCAC | 版权登记信息 [ncac.gov.cn](https://www.ncac.gov.cn/) |
| 国际版权公约 | 伯尔尼公约 / TRIPS / WCT |

⚠️ 字体版权警告：包装上的字体可能构成独立版权。**查不到 ≠ 无版权**。

## ✅ 自查清单 | Self-Check List

14项专业检测项（按四步法组织）：

**专利 Patent (3项)：**
1. 外观设计专利检索 Design patent search
2. 洛迦诺分类筛选 Locarno classification filtering
3. 第三方专利工具复核 Third-party patent tool verification

**版权 Copyright (3项)：**
4. 作品登记公告查询 Copyright registration search
5. 字体版权确认 Font copyright verification
6. 国际版权公约保护确认 International copyright convention check

**商标 Trademark (4项)：**
7. 商标检索（含立体商标）Trademark search (incl. 3D trademarks)
8. 尼斯分类排查 Nice classification screening
9. 国际商标排查 International trademark screening
10. 反不正当竞争法保护评估 Unfair competition protection assessment

**图像比对 Image Comparison (4项)：**
11. 以图搜图（6+平台）Reverse image search (6+ platforms)
12. 电商平台搜同款 E-commerce same-product search
13. 小红书/抖音搜同款 Xiaohongshu/Douyin search
14. 设计社区排查 Design community screening

- ✅ 勾选进度追踪（localStorage 持久化）
- 📄 导出检测报告（含产品信息、检测记录、结论、建议）
