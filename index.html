1 查外观设计专利 最关键

操作步骤

  1. 进入国知局专利检索系统，选择「高级检索」
  2. 选择「外观设计」分类
  3. 填写产品名称、用途（如"包装盒""卫生巾包装"）
  4. 按洛迦诺分类筛选——包装多为 09类（09-01包装容器、09-03盒/箱等）
  5. 上传包装正/侧/顶视图，系统自动比对相似度
  6. 导出高相似列表，逐一排查

重点看什么

  • 相同/近似设计 — 整体观察、综合判断，不是比细节，是比整体印象
  • 申请日 — 比你的设计更早还是更晚
  • 法律状态 — 授权才构成权利，申请中/驳回/无效的不构成
  • 注意：虚线部分不受保护，实线部分才受保护

2 查著作权（版权）

操作步骤

  1. 进入中国版权保护中心 → 作品登记公告查询
  2. 输入检索词：包装名称、设计方、著作权人、标注信息
  3. 查看登记证书上的作品名称、登记日期、著作权人

重要说明

  • 版权是自动保护的，登记不是必需条件
  • 查不到登记 ≠ 无版权，很多作品未登记也受保护
  • 但查到登记可佐证权属，对维权和抗辩都有利
  • 版权保护的是表达形式，不保护思想/功能

国际版权相关公约

  • 《伯尔尼公约》 — 181国加入，版权自动保护的国际基础，中国1992年加入
  • 《世界版权公约》 — UCC，与伯尔尼公约互补
  • 《TRIPS协定》 — WTO框架下知识产权保护最低标准
  • 《WIPO版权条约》(WCT) — 数字环境下的版权保护

中国是以上所有公约的缔约国，在国内登记的版权在成员国间互认保护

3 查商标（含立体商标）

检索方式

  • 文字商标 — 查品牌名是否已被注册
  • 图形商标 — 查包装上的图形标识是否近似
  • 立体商标 — 查瓶型/盒型的立体形状是否已注册
  • 核对 相同/近似 + 同类商品（需查全相关类别）

知名品牌特有包装/装潢

即使没有专利/商标，知名品牌特有的包装装潢也受《反不正当竞争法》保护。例：可口可乐瓶型、LV花纹、茅台酒瓶——包装形状、颜色组合、图案风格如果与知名品牌高度近似，即使无专利也构成侵权。

中国商标法 — 包装相关全品类分类（尼斯分类）

包装设计需同时查以下所有相关类别，跨类近似也可能构成侵权：

  • 第1类 — 化学原料、化学制剂（消毒剂、净水剂等）
  • 第3类 — 化妆品、洗护用品、清洁剂
  • 第5类 — 医药、卫生用品（卫生巾、护理用品、医用敷料）
  • 第8类 — 手工器具（美妆工具等）
  • 第9类 — 科学仪器、APP软件
  • 第10类 — 医疗器械
  • 第16类 — 纸制品、包装材料、印刷品、纸巾
  • 第20类 — 家具、容器（非金属包装容器）
  • 第21类 — 家用容器、化妆用具、梳妆盒
  • 第24类 — 纺织品、布料（包装袋/布袋）
  • 第29类 — 食品（罐头、密封包装食品）
  • 第30类 — 调味品、方便食品
  • 第35类 — 广告、商业管理（品牌服务类保护）
  • 第40类 — 材料加工（印刷、包装加工）

国际商标法 — 主要体系

  • 马德里体系 — 一次申请，130+成员国生效，中国1995年加入
  • 欧盟商标(EUTM) — 一次注册，27个欧盟成员国有效
  • 美国(Lanham Act) — 使用在先原则，需提供使用证据
  • 东盟(ASEAN) — 各国独立注册，无统一体系
  • 非洲(OAPI/ARIPO) — 区域性商标体系

出口产品必须查目标国商标，国内注册≠海外受保护

创作证据留存指引

原创性的关键不仅是"做了什么"，更是"能证明什么时候做的"。以下工具和平台可帮你建立时间戳证据链。

⚡ 区块链存证
上传设计稿获取不可篡改的时间戳，几分钟即可完成，费用极低（几元/次）
阿里云可信存证 更多平台
📜 著作权登记
中国版权保护中心登记，法律直接推定权属，30-60天出证，费用300-500元
中国版权保护中心 登记档案查询
🏛️ 公证
对设计稿做公证，法律效力最高，但费用较高（1000-3000元），适合高价值设计
中国法网（查公证处） 在线公证平台
📧 邮件/消息时间戳
设计完成即发送确认邮件或飞书消息，作为辅助时间证据（不单独成立，但可形成证据链）
🗂️ 版本管理归档
使用 Git 管理设计文件版本，每次 commit 都有不可篡改的时间戳（适合技术型团队）
📋 设计过程文档
保留草图、修改记录、会议纪要、客户沟通记录，形成完整的创作过程证据链

最佳实践

1. 设计完成即存证 — 不要等到有纠纷才存证，每个版本都应该有时间戳

2. 重要项目做著作权登记 — 费用不高，法律效力强

3. 过程文档不要删除 — 草图、修改记录、沟通记录都是辅助证据

4. 出口产品查目标国专利 — 国内原创不代表海外无冲突

5. 字体/素材授权留底 — 授权协议截图+付款凭证都要保存
